Configura la sincronización

Para que todos los miembros de la familia vean los mismos datos en tiempo real, necesitas una cuenta gratuita en JSONBin.io. Solo hay que hacerlo una vez. El resto de la familia solo necesita la clave que tú les des.

Paso 1 — Ve a jsonbin.io → Crear cuenta (gratis)
Paso 2 — En el panel ve a API Keys → copia tu Master Key (empieza por $2b$10$…) o una Access Key
Paso 3 — Selecciona el tipo de clave abajo, pégala y haz clic en "Conectar"
Se guarda solo en tu navegador, nunca se envía a ningún servidor nuestro.
Si es la primera vez, déjalo vacío — lo crearemos automáticamente. Si ya tienes uno, pégalo aquí para compartir los mismos datos.

Error al conectar. Revisa que la clave sea correcta.

También puedes usar localmente sin sincronización y conectar más tarde desde Configuración. Los datos gratis en JSONBin están limitados a 10.000 peticiones/mes y 10 MB, más que suficiente para uso doméstico.

⚙ Configuración de sincronización

Los datos se sincronizan mediante JSONBin.io (servicio gratuito). Guarda tu Master Key y Bin ID para poder acceder desde otros dispositivos o compartir con tu familia.

Master Key → acceso completo, puede crear bins nuevos. Úsala en tu dispositivo principal.
Access Key → más segura para compartir, solo lee/escribe un bin concreto. Requiere Bin ID existente.
Ambas se encuentran en jsonbin.io → API Keys
Se guarda solo en tu navegador (localStorage). Nunca se envía a ningún servidor nuestro.
Comparte este ID con tu familia para que todos usen los mismos datos.
